# Verificador de Habilitação
## O programa solicita o nome e a idade do usuário e retorna uma resposta baseada na legislação brasileira:
 Um script simples em Python que interage com o usuário para verificar se ele já tem idade suficiente
para iniciar o processo de tirar a Carteira Nacional de Habilitação (CNH).

🚀 **Funcionalidades:**  
O programa solicita o nome e a idade do usuário e retorna uma resposta baseada na 
legislação brasileira:  
- 18 anos ou mais: Liberado para tirar a habilitação.
- 17 anos: Pode iniciar as aulas na autoescola.
- Menos de 17 anos: Ainda não tem idade permitida.
